
# 🐳 Atividade 2.1 : Configuração do Ambiente de Trabalho com o GitLab

Este laboratório prático marcou o início das atividades operacionais do Módulo 2, focando na instalação, provisionamento e administração de um servidor local próprio para controle de versão e governança.

## 🛠️ O que foi aprendido e implementado:
* **Instalação da Infraestrutura:** Criação e inicialização de um servidor completo do GitLab rodando de forma isolada dentro de um contêiner **Docker**.
* **Versionamento de Código:** Inicialização do ecossistema Git local e importação de uma aplicação web em **Golang** (`my-golang-app`) para o repositório remoto.
* **Administração via CLI (Linha de Comando):** Automação de tarefas administrativas, criando grupos corporativos e provisionando usuários de teste diretamente pelo terminal de comandos.
* **Administração via GUI (Interface Gráfica):** Gerenciamento fino de credenciais, auditoria visual de membros e governança de acessos na plataforma.
* **Associação e RBAC:** Vinculação hierárquica do grupo de usuários ao repositório do projeto, estabelecendo papéis e permissões de segurança padronizadas (**Maintainer**).
