### 🛡️ Módulo 2: Infraestrutura Ágil e Pipeline Lifecycle

#### 🧪 Laboratório 2.2: Operação do Ambiente de Trabalho com GitLab
Neste laboratório prático focado em governança e operações ágeis, assumi o papel de administradora de infraestrutura de controle de versão.

* **Administração Baseada em API (CLI):** Utilização prática do utilitário `curl` para interagir com os *endpoints* da API REST do GitLab, realizando auditoria de usuários cadastrados e automação de privilégios via requisições `GET` e `PUT` para gerenciamento de RBAC.
* **Estratégia de Ramificação (Git Flow):** Criação e isolamento de novas funcionalidades em branches secundárias (`feature01`), execução de commits estruturados, transição de ambientes e fusão segura (*merge*) de volta à ramificação principal (`master`).
* **Sincronização e Rastreabilidade:** Utilização do comando `git log --graph` para validação visual da árvore de commits diretamente no terminal e sincronização remota via `git push`.
* **Cultura Ágil (Gestão Visual):** Implementação de gerenciamento de tarefas através do módulo de *Issues* do GitLab para rastreamento de progresso e demandas de projeto.
